### Lab 3: Leukemia Classification with ResNet18, 50, 152

📌任務目標: 實作並比較不同深度的 ResNet (Residual Network) 架構（18, 50, 152 層），用於辨識白血球影像以診斷急性淋巴性白血病（Leukemia）

📊Dataset:
- 白血球疾病辨識，主要是由白血球外觀判斷此病患是否有白血病(血癌)
- 圖片檔和其對應 label 的 csv 檔

🛠️核心技術:
- 殘差區塊 (Residual Block):
  a. BasicBlock (用於 ResNet18)
    - 由兩個 3 x 3 的卷積層組成
    - 結構簡單直觀，適合層數較淺的網路
  b. Bottleneck (用於 ResNet50 / 152)
    - 採用 1 x 1 → 3 x 3 → 1 x 1 的三層組合
    - 設計動機:
      1. 降維: 第一個 1 x 1 卷積負責減少通道數（壓縮資訊），降低後續 3 x 3 卷積的計算負擔
      2. 特徵提取: 中間的 3 x 3 卷積在低維空間進行特徵運算
      3. 升維：最後一個 1 x 1 卷積將通道數還原，以便與 Shortcut 路徑相加
    - 優勢: 在相同運算資源下，Bottleneck 能讓模型疊得更深而不爆炸，是深層 ResNet 成功的關鍵

- 殘差路徑 (Shortcut Connection)
  - 傳統卷積網路在疊加到上百層時，梯度在反向傳播過程中會因為連鎖效應不斷相乘，最終趨近於 0（梯度消失）
  - ResNet 引入了 $H(x) = F(x) + x$ 的概念。 $F(x)$ 是卷積層學習到的殘差，而 $x$ 是原始輸入
  - 在求導時，$\frac{\partial (F(x) + x)}{\partial x} = \frac{\partial F(x)}{\partial x} + 1$
  - 這個 「+1」 就像一條梯度高速公路，確保即使卷積層（殘差部分）的梯度很小，整體的梯度依然能維持在 1 附近，讓底層權重能確實收到更新訊號
  - 維度對齊：如果 $F(x)$ 改變了通道數或圖片尺寸（Downsample），Shortcut 路徑會補上一層 1 x 1 卷積，確保 $F(x)$ 與 $x$ 能夠在數學上正確相加

- Adaptive Pooling
  - 在傳統 CNN 中，最後接全連接層（FC）前，必須知道精確的矩陣尺寸（例如 7 x 7 x 512）。這導致模型一旦換了輸入圖片大小（如從 224 改成 256），程式就會因維度不符而報錯
  - 自動適應: 不同於標準 Pooling 需要指定 kernel_size 和 stride，Adaptive Pooling 只需要指定「輸出的尺寸」（例如 AdaptiveAvgPool2d(1)），它會自動計算卷積核大小，強行將輸出壓縮成你指定的 1 x 1
  - 空間靈活性: 這讓 ResNet 可以接受不同解析度的白血球影像，而不必每次都為了對齊尺寸而導致影像比例失真
  - 減少參數量: 取代了傳統巨大的全連接層前期處理，讓接在後面的 Linear 層（FC）Input 維度固定，大幅簡化架構設計

💡實驗發現:
- 解決梯度消失: ResNet 的核心在於殘差路徑，它讓梯度可以跳過部分網路層直接往前傳遞，有效解決了深層網路容易遇到的梯度消失問題
- 隨意壓縮圖片會失去原先的部分特徵（如白血球核的紋理），若損失的是辨識病徵的重要細節，模型結果會大幅變差
